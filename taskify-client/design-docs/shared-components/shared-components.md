# Shared Components

Components that appear across multiple pages in this project.

## Core Form Components

- **button** (standard) - Used across all pages and dialogs
- **input** (standard) - Used in all forms
- **label** (standard) - Used in all forms
- **card** (standard) - Used in layouts and pages

## Dialog Components

- **dialog** (standard) - Used in all dialogs
- **form** (validation) - Used in create/update dialogs

## Navigation

- **navigation-menu** (standard) - Used in MainLayout
- **avatar** (standard) - Used in MainLayout for user display

## Data Display

- **table** (standard) - Used in HomePage
- **badge** (standard) - Used in HomePage and ViewTaskPage
- **separator** (standard) - Used in layouts and pages

## Form Inputs

- **textarea** (standard) - Used in task dialogs
- **calendar** (standard) - Used in task dialogs for date picker
- **checkbox** (standard) - Used in UpdateTaskDialog
- **select** (standard) - Used in HomePage for filters

## Specialized

- **pagination** (standard) - Used in HomePage
- **alert-dialog** (destructive) - Used in DeleteTaskDialog
